3. Compare and contrast mutable and immutable objects in Python with examples.
-> Mutable Objects:
Definition : Objects whose state can be changed after creation.
Modifications : Modify in place without creating a new object.
Memory: Generally more efficient in memory as they don't require creating new 
objects for modifications.
Memory Address : Remains the same even after modification.
Examples : Lists, dictionaries, sets.
Immutable Objects:
Definition: Objects whose state cannot be changed after creation.
Examples: Strings, tuples, frozensets, numbers.
Modifications: Creating a new object is required for any modification.
Memory: May use more memory due to creating new objects for each modification.
Memory Address: Changes when a new value is assigned.
Examples: Strings, tuples, frozensets, numbers.

functions like int() float() str() bool().
Ex - # int to float
a = 10
b = float(a)
print(b)        # Output: 10.0

7. Describe the different types of loops in Python and their use cases with 
examples.
-> In Python, loops are used to execute a block of code repeatedly. There are 
two main types:
1. for Loop
Used to iterate over a sequence (like a list, tuple, string, or range).
Ex - Iterating through a list:
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)
2. while Loop
Ex - count = 0
while count < 5:
Use Case - Best for looping when you know the number of iterations or when 
iterating over a collection.
    print(count)
    count += 1
Use case: Useful when the number of iterations is unknown and depends on a 
condition being met.
  1. Describe the different types of loops in Python and their use cases with examples.

-> In Python, loops are used to execute a block of code repeatedly. There are two main types:

  1. for Loop Used to iterate over a sequence (like a list, tuple, string, or range). Ex - Iterating through a list: fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit) Use Case - Best for looping when you know the number of iterations or when iterating over a collection.

  2. while Loop Ex - count = 0 while count < 5: print(count) count += 1 Use case: Useful when the number of iterations is unknown and depends on a condition being met.
